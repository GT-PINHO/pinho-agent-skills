# Example 03 — Proof before CTA

## Input (user)

Landing page has CTA above the fold but no testimonials, no numbers, no case proof.  
Traffic: cold prospecting  
CTR: 1.3%  
CVR: 1.1%

## Expected output (assistant)

**1) Diagnosis**
- Dominant bottleneck: lack of proof for cold traffic
- Evidence: cold prospecting + no proof + CVR ~1%

**2) Fix list (prioritized)**
- P0: add proof block above the primary CTA (case, numbers, before/after)
- P1: add objection handling section (price, time, skepticism)
- P1: tighten hero mechanism (why you, why now)

**3) Test plan**
- Change first: proof block + mechanism clarity
- Measure: scroll depth to proof, CTA clicks, CVR
- Time window: 7 days (cold traffic needs volume)

**4) Validation**
- Leading indicators: CTA click-to-form start up, bounce down
- Lagging metric: CVR to ≥ 2–3% (or higher depending on offer)
- Kill rule: if CVR stays < 1.5% after proof, audit offer/ICP
